# Free Septic Resources Distribution Plan

Primary hub:

`https://getmaintane.com/free-septic-resources.html`

Use this page when the audience is broad, cold, or not ready for a product pitch. It should be the default link for resource-page outreach, social bios, community answers, and creator references because it lets people self-select into checklist, printable, symptom, home-type, and product-education paths.

## Positioning

Maintane publishes simple septic resources for homeowners who want fewer surprises and less guesswork.

Core promise:

`Free septic checklists, printables, and homeowner guides in one place.`

Support points:

- Practical before promotional.
- Clear enough for new septic homeowners.
- Useful for rentals, cabins, Airbnbs, and family homes.
- Honest about when to call a septic professional.
- Bridges naturally into Maintane dosing and product education.

- `page_view` for `/free-septic-resources.html`.
- `nav_click` from footer links into the hub.
- `lead_form_submit` from the hub form.
- `product_cta_click` after hub visits.
- Assisted traffic paths: hub -> checklist -> dosing guide -> product page.

Watch these in Search Console:

- Queries containing `free septic checklist`.
- Queries containing `septic resources`.
- Queries containing `printable septic`.
- Queries containing `septic rules for guests`.
- Queries containing `septic homeowner guide`.

## When To Improve The Hub

Improve title/meta if impressions rise but CTR is weak.

Add a new section if Search Console shows repeated queries that do not fit the current buckets.

Do not add unrelated product claims. The hub should stay practical and resource-first.
